## Project Overview
The AR Heritage App is an Augmented Reality (AR) based web application designed to provide an interactive platform for exploring cultural and historical heritage sites. The system allows users to visualize heritage content using 3D models, multimedia integration, and smart travel features such as map-based navigation, hotel discovery, and ticket booking.

The application enhances learning and virtual tourism by combining immersive visualization with a user-friendly interface.

## Installation & Setup

### **1. Clone the Repository**

```bash
git clone https://github.com/RiyaJayanti/AR_Heritage_App.git
```

---

### **2. Navigate to Project Folder**

```bash
cd AR_Heritage_App
```

---

### **3. Run the Frontend (Web Interface)**

- Install **Visual Studio Code**
- Install the **Live Server** extension
- Open the project folder in VS Code
- Right click `index.html`
- Click **Open with Live Server**

The application will open in your browser automatically.

---

### **4. Run Backend (Gesture Recognition & Processing)**

Install required dependencies:

```bash
pip install opencv-python mediapipe numpy
```

Run the backend script:

```bash
python main.py
```
